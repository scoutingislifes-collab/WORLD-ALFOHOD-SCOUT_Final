# Global Leopard City - مدينة الفهود العالمية
### نظام كشفي دامج (للأسوياء وذوي الهمم)
## طريقة التشغيل:
1. افتح المجلد في Antigravity أو VSCode.
2. اكتب في Terminal: 
pm install
3. ابدأ المشروع بأمر واحد: 
pm run dev
## المميزات:
- واجهة احترافية تدعم ذوي الهمم.
- نظام إدارة كشفي متكامل (Frontend & Backend).
- متوافق تماماً مع بيئة Replit و Antigravity.
